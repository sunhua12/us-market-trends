# US Market Trends Dashboard

這是一個模仿 `github-trends` 架構的美國市場熱度分析儀表板。

## 🚀 核心功能
- **數據抓取**：自動從 Yahoo Finance 抓取全球主要指數與最活躍的美國股票數據。
- **自動化更新**：透過 `trigger.sh` 一鍵更新數據並重新生成儀表板。
- **現代化介面**：基於 Bootstrap 5 的簡潔響應式設計。

## 📂 專案架構
```text
us-market-trends/
├── scripts/
│   ├── fetch_market_data.py   # 市場數據抓取腳本
│   └── generate_dashboard.py  # 儀表板生成腳本
├── assets/
│   └── dashboard_template.html # HTML/Jinja2 模板
├── data/
│   └── market_data.json       # 原始數據存儲
├── index.html                 # 最終生成的儀表板
└── trigger.sh                 # 全自動執行腳本
```

## 📦 使用方法
1. **執行更新**：
   ```bash
   ./us-market-trends/trigger.sh
   ```
2. **查看結果**：
   在瀏覽器中打開 `us-market-trends/index.html`。
