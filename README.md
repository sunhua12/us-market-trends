[![Live Dashboard](https://img.shields.io/badge/Live-Dashboard-blue?style=for-the-badge&logo=github)](https://sunhua12.github.io/us-market-trends/)

一個專為美國市場熱度設計的現代化儀表板。結合了市場即時數據與社群討論分析，提供極具視覺衝擊力的深色模式與圖表分析。

## 🚀 核心功能
- **市場視覺化**：整合 **ApexCharts**，直觀呈現熱門股票漲跌幅分佈。
- **社群情緒分析**：自動抓取 **Reddit** (`r/wallstreetbets`, `r/stocks`, `r/investing`) 的實時熱點討論。
- **自動化更新**：透過 **GitHub Actions** 每日定時（美股開盤前）抓取數據，並發佈至 GitHub Pages。
- **現代化設計**：類玻璃擬態 (Glassmorphism) 深色模式，專為高密度資訊閱讀優化。

## 📦 如何使用
### 本地開發
1. 確保已安裝 Python 依賴：
   ```bash
   pip install httpx beautifulsoup4 jinja2
   ```
2. 執行本地腳本更新數據：
   ```bash
   ./us-market-trends/trigger.sh
   ```

### 線上 Trigger
1. 前往您的 GitHub Repository 頁面。
2. 點擊 **Actions** 分頁。
3. 選擇 **Update US Market Dashboard**。
4. 點擊 **Run workflow** 手動觸發即時更新。
